# 基本操作

## 1. 函数调用与对象识别

在 R 语言中，区分“执行指令”与“查看对象”的关键在于 **括号 `()`**。

* **执行函数**：函数名后必须跟随括号。
  * 示例：`ls()` 用于列出当前环境中的所有对象。
* **查看对象**：若不带括号，R 会将其视为一个对象（Object）并尝试显示其内容或源代码。
  * 示例：输入 `ls` 将显示该函数的内部代码，而非执行操作。

## 2. 工作目录 (Working Directory) 管理

工作目录是 R 运行时的默认“home目录”，所有未指定路径的文件读写操作均在此进行。

* **获取路径**：`getwd()` —— 查看当前 R 正在哪个文件夹下工作。
* **设置路径**：`setwd("path/to/folder")` —— 切换工作目录。
* **文件检索**：`list.files()` 或 `dir()` —— 查看当前目录下的所有文件和子文件夹。

## 3. 赋值符号 (Assignment Operators)

R 语言拥有其独特的赋值体系，建议养成良好的输入习惯。

| 符号    | 类型       | 说明                            | 快捷键 (RStudio) |
| ----- | -------- | ----------------------------- | ------------- |
| `<-`  | **标准赋值** | 最常用的赋值方式，用于定义局部变量。            | `Alt` + `-`   |
| `<<-` | **全局赋值** | 强制赋值给全局变量，常用于函数内部修改外部状态。      | -             |
| `=`   | **参数传递** | 虽然也可用于赋值，但在 R 中主要用于函数内部的参数指定。 | -             |

> **注意**：建议始终使用 `<-` 进行变量赋值，以避免在复杂的函数作用域中出现难以排查的“玄学错误”。

## 4. tips

R 控制台是一个强大的交互式计算器。

* **直接运算**：在控制台输入表达式后回车，直接返回结果。
* **结果存储**：可以将运算结果保存到变量中供后续调用。
* 计算 1, 2, 3 的平均值时，应使用向量形式：
  * `mean(c(1, 2, 3))` —— 正确写法，结果为 `2`。
  * `y <- mean(c(1, 2, 3))` —— 等价于执行 `y <- 2`。
